# Stock Trading Platform 📈

Stock Trading Platform is a Java Swing-based desktop application developed to simulate real-world stock market trading. The application allows users to buy and sell stocks, manage their portfolio balance, track transaction history, and monitor profit/loss through an interactive graphical user interface.

## Features
- Secure login system
- Buy stocks
- Sell stocks
- Portfolio balance management
- Transaction history using JTable
- Real-time profit/loss tracking
- Dashboard for balance and shares
- User-friendly GUI using Java Swing

## Technologies Used
- Java
- Java Swing
- JTable
- Object-Oriented Programming (OOP)
- Event Handling

## How It Works
1. Login using valid credentials.
2. Select a stock from available options.
3. Enter quantity of shares.
4. Buy or sell stocks.
5. System automatically updates:
   - Balance
   - Total Shares
   - Profit/Loss
   - Transaction History
